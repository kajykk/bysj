# v1.5 API 文档 - 性能监控与可观测性 (已废弃)

## 概述

本文档描述 v1.5 版本新增的 API 接口，包括监控指标采集、灰度发布管理、离线验证、报告导出等功能。

## 监控指标 API

### 1. 获取系统指标

获取当前系统性能指标。

```http
GET /monitoring/metrics
```

**响应参数**:

| 参数 | 类型 | 说明 |
|------|------|------|
| model_success_rate | float | 模型成功率 (0-1) |
| fallback_rate | float | 回退率 (0-1) |
| avg_latency_ms | float | 平均延迟 (ms) |
| p99_latency_ms | float | P99 延迟 (ms) |
| request_count | int | 请求总数 |
| error_count | int | 错误总数 |
| active_models | int | 活跃模型数 |
| timestamp | string | ISO 8601 时间戳 |

**响应示例**:

```json
{
  "model_success_rate": 0.985,
  "fallback_rate": 0.015,
  "avg_latency_ms": 145,
  "p99_latency_ms": 320,
  "request_count": 12580,
  "error_count": 189,
  "active_models": 3,
  "timestamp": "2024-01-15T10:30:00Z"
}
```

### 2. 获取历史指标

获取指定时间范围内的历史指标数据。

```http
GET /monitoring/metrics/history
```

**查询参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| start_time | string | 是 | 开始时间 (ISO 8601) |
| end_time | string | 是 | 结束时间 (ISO 8601) |
| interval | string | 否 | 聚合间隔: 1m, 5m, 1h, 1d (默认: 5m) |

**响应示例**:

```json
{
  "data": [
    {
      "timestamp": "2024-01-15T10:00:00Z",
      "model_success_rate": 0.98,
      "request_count": 1200
    }
  ]
}
```

### 3. 上报前端性能指标

前端性能监控数据上报端点。

```http
POST /monitoring/frontend-metrics
```

**请求体**:

| 参数 | 类型 | 说明 |
|------|------|------|
| fcp | float | First Contentful Paint (ms) |
| lcp | float | Largest Contentful Paint (ms) |
| fid | float | First Input Delay (ms) |
| cls | float | Cumulative Layout Shift |
| ttfb | float | Time to First Byte (ms) |
| page_load_time | float | 页面完全加载时间 (ms) |
| url | string | 页面 URL |

## 灰度发布 API

### 4. 创建灰度发布

创建新的灰度发布任务。

```http
POST /canary/deployments
```

**请求体**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| baseline_version | string | 是 | 基线版本号 |
| canary_version | string | 是 | 灰度版本号 |
| traffic_percent | float | 是 | 灰度流量比例 (0-100) |
| auto_rollback | boolean | 否 | 是否启用自动回滚 (默认: true) |
| success_threshold | float | 否 | 成功率阈值 (默认: 0.98) |

**响应示例**:

```json
{
  "id": "deploy_12345",
  "status": "running",
  "baseline_version": "v1.0.0",
  "canary_version": "v2.0.0",
  "traffic_percent": 10,
  "created_at": "2024-01-15T10:30:00Z"
}
```

### 5. 获取灰度发布列表

```http
GET /canary/deployments
```

**查询参数**:

| 参数 | 类型 | 说明 |
|------|------|------|
| status | string | 过滤状态: running, success, failed, rolled_back |
| page | int | 页码 (默认: 1) |
| page_size | int | 每页数量 (默认: 10) |

### 6. 调整灰度流量

```http
PATCH /canary/deployments/{deployment_id}
```

**请求体**:

| 参数 | 类型 | 说明 |
|------|------|------|
| traffic_percent | float | 新的灰度流量比例 (0-100) |

### 7. 回滚灰度发布

```http
POST /canary/deployments/{deployment_id}/rollback
```

**响应示例**:

```json
{
  "id": "deploy_12345",
  "status": "rolled_back",
  "traffic_percent": 0,
  "rolled_back_at": "2024-01-15T10:35:00Z"
}
```

### 8. 完成灰度发布（全量发布）

```http
POST /canary/deployments/{deployment_id}/promote
```

## 离线验证 API

### 9. 创建验证任务

创建离线模型验证任务。

```http
POST /validation/tasks
```

**请求体**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| model_version | string | 是 | 模型版本号 |
| dataset_id | string | 是 | 验证数据集 ID |
| metrics | array | 否 | 指定评估指标 (默认: 全部) |

**响应示例**:

```json
{
  "task_id": "val_12345",
  "status": "pending",
  "model_version": "v2.0.0",
  "created_at": "2024-01-15T10:30:00Z"
}
```

### 10. 获取验证结果

```http
GET /validation/tasks/{task_id}
```

**响应示例**:

```json
{
  "task_id": "val_12345",
  "status": "completed",
  "model_version": "v2.0.0",
  "metrics": {
    "accuracy": 0.87,
    "precision": 0.86,
    "recall": 0.88,
    "f1_score": 0.87,
    "auc_roc": 0.93,
    "auc_pr": 0.91
  },
  "confusion_matrix": {
    "tp": 450,
    "fp": 50,
    "tn": 430,
    "fn": 70
  },
  "completed_at": "2024-01-15T10:35:00Z"
}
```

### 11. 获取漂移检测结果

```http
GET /validation/drift
```

**查询参数**:

| 参数 | 类型 | 说明 |
|------|------|------|
| model_version | string | 模型版本号 |
| window | string | 时间窗口: 1h, 6h, 24h, 7d |

## 报告导出 API

### 12. 生成 PDF 报告

```http
POST /reports/pdf
```

**请求体**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| report_type | string | 是 | 报告类型: user_risk, counselor, admin |
| start_date | string | 是 | 开始日期 (YYYY-MM-DD) |
| end_date | string | 是 | 结束日期 (YYYY-MM-DD) |
| user_id | int | 否 | 用户 ID (user_risk 类型必填) |

**响应示例**:

```json
{
  "task_id": "rpt_12345",
  "status": "processing",
  "format": "pdf",
  "estimated_completion": "2024-01-15T10:35:00Z"
}
```

### 13. 生成 Excel 报告

```http
POST /reports/excel
```

**请求体**: 同 PDF 报告

### 14. 获取导出任务状态

```http
GET /reports/tasks/{task_id}
```

**响应示例**:

```json
{
  "task_id": "rpt_12345",
  "status": "completed",
  "format": "pdf",
  "file_size": 2621440,
  "download_url": "/api/v1/reports/download/rpt_12345",
  "completed_at": "2024-01-15T10:35:00Z"
}
```

### 15. 下载报告文件

```http
GET /reports/download/{task_id}
```

**响应**: 文件流 (Content-Type: application/pdf 或 application/vnd.openxmlformats-officedocument.spreadsheetml.sheet)

### 16. 获取导出历史

```http
GET /reports/history
```

**查询参数**:

| 参数 | 类型 | 说明 |
|------|------|------|
| report_type | string | 报告类型过滤 |
| format | string | 格式过滤: pdf, excel |
| status | string | 状态过滤 |
| page | int | 页码 |
| page_size | int | 每页数量 |

## 错误码

| 状态码 | 错误码 | 说明 |
|--------|--------|------|
| 400 | INVALID_REQUEST | 请求参数错误 |
| 401 | UNAUTHORIZED | 未授权 |
| 403 | FORBIDDEN | 权限不足 |
| 404 | NOT_FOUND | 资源不存在 |
| 409 | CONFLICT | 资源冲突 (如灰度发布已存在) |
| 422 | VALIDATION_ERROR | 验证失败 |
| 429 | RATE_LIMITED | 请求过于频繁 |
| 500 | INTERNAL_ERROR | 服务器内部错误 |
| 503 | SERVICE_UNAVAILABLE | 服务不可用 |

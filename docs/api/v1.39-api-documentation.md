# v1.39 API 文档 (完整端点清单)

本文档列出 Depression Warning System v1.39 所有 HTTP API 端点, 按 router (tag) 分组. 端点详细参数/请求体/响应 schema 请查阅 OpenAPI 规范 (Swagger UI: `/docs`).

**版本**: v1.39 (应用内部版本 3.1.0)  
**基础路径**: `/api/v1` (v1 router) / 顶层路径 (csp_report/analytics)  
**认证方式**: Bearer Token (JWT, 部分 public 端点豁免)  
**OpenAPI 规范**: [openapi.json](../../backend/tests/contract/openapi.json) (58054 行, 132 路径)

> 本文档由 MAINT-P1-002 任务基于 openapi.json 自动生成, 替代废弃的 [v1.5-api-documentation.md](./v1.5-api-documentation.md).

## 认证说明

- **JWT Bearer Token**: 在 `Authorization` 头中携带 `Bearer <access_token>`
- **公开端点**: `/api/v1/auth/login`, `/api/v1/auth/register`, `/health*`, `/metrics`, `/csp-report`, `/api/v1/version` 等豁免认证
- **角色权限**: 部分端点要求特定角色 (admin/counselor/user), 通过 `Depends(require_role(...))` 校验
- **SEC-P1-001**: JWT blocklist 支持令牌失效 (登出后 token 立即失效)

## 限流策略

| 层级 | 路径模式 | 速率 | burst |
|------|---------|------|-------|
| nginx | `/api/` | 10 r/s | 20 |
| nginx | `/api/v1/auth/` | 10 r/m | 10 |
| nginx | `/api/v1/auth/refresh` | 20 r/m | 5 |
| slowapi | PDF/Excel 生成, ML 实验, 校验运行 | 5/minute | - |
| slowapi | 金丝雀部署/回滚/pause/resume/traffic | 10/minute | - |
| slowapi | 普通查询 (列表/详情/templates) | 30/minute | - |
| slowapi | 健康检查端点 (`/health/*`) | 豁免 | - |

## 错误响应格式

所有错误响应使用统一格式 (STAB-P1-001 响应体统一):

```json
{
  "detail": {
    "code": "ERROR_CODE",
    "message": "人类可读错误信息"
  }
}
```

常见错误码:

| HTTP 状态码 | code | 说明 |
|------|------|------|
| 400 | BAD_REQUEST | 请求参数错误 |
| 401 | UNAUTHORIZED | 未认证或 token 失效 |
| 403 | FORBIDDEN | 无权限 |
| 404 | NOT_FOUND | 资源不存在 |
| 409 | CONFLICT | 资源冲突 |
| 422 | UNPROCESSABLE_ENTITY | 请求体验证失败 |
| 429 | RATE_LIMITED | 限流触发 |
| 503 | SERVICE_UNAVAILABLE | 熔断器 OPEN 或服务不可用 |

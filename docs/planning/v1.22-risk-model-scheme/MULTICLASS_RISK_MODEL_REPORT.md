# v1.22 五级风险方案报告 (MULTICLASS_RISK_MODEL_REPORT)

> **日期**: 2026-05-01  
> **迭代**: v1.22-risk-model-scheme  
> **报告类型**: 方案替代版  
> **依据**: v1.21 五级风险模型失败分析

---

## 一、背景

v1.21 已尝试基于派生五级标签训练独立五分类模型，但结果表明：当前数据条件下，该方向不具备上线可行性。

因此，v1.22 不再把“训练独立五级风险模型”作为主路径，而改为制定“**五级风险生成方案**”。

---

## 二、v1.21 失败事实回顾

### 2.1 LogisticRegression Multinomial

| 指标 | 结果 |
|------|------|
| Accuracy | 0.1333 |
| F1 Macro | 0.1365 |
| High/Critical Recall | 0.6154 |

### 2.2 RandomForest

| 指标 | 结果 |
|------|------|
| Accuracy | 0.3200 |
| F1 Macro | 0.1921 |
| High/Critical Recall | 0.1154 |

上述结果远低于可接受水平，说明当前独立五分类建模不成立。

---

## 三、失败原因归纳

| 原因 | 说明 |
|------|------|
| 标签为派生标签 | Five-Level 不是天然标注，而是规则生成 |
| 泄漏约束严格 | 为避免泄漏必须移除构造标签的关键字段 |
| 基础特征信号弱 | 剩余 10 个特征无法区分 5 个等级 |
| 样本分布稀疏 | Moderate / Critical 样本数不足 |
| 任务定义不稳 | 风险等级更适合概率分档而非硬分类 |

---

## 四、v1.22 替代方案

### 4.1 方案核心

v1.22 将五级风险定义为“**二分类风险概率的业务映射结果**”，而不是“独立训练得到的五分类标签”。

生成链路如下：

```text
结构化输入特征
→ 二分类模型输出 Depression Probability
→ 概率校准 / 风险分数映射
→ 五级风险等级
→ 展示 / 告警 / 复核动作
```

### 4.2 等级定义

| 编码 | 等级 | 概率区间 | 业务含义 |
|------|------|----------|----------|
| 0 | None | p < 0.15 | 无明显风险信号 |
| 1 | Mild | 0.15 ≤ p < 0.35 | 轻度关注 |
| 2 | Moderate | 0.35 ≤ p < 0.65 | 中度风险，建议进一步关注 |
| 3 | High | 0.65 ≤ p < 0.85 | 高风险，建议人工复核 |
| 4 | Critical | p ≥ 0.85 | 极高风险，建议优先复核/升级处理 |

## 五、相对独立五分类模型的优势

| 维度 | 独立五分类 | v1.22 概率分档方案 |
|------|------------|--------------------|
| 训练可行性 | 差 | 高 |
| 数据需求 | 高 | 中 |
| 泄漏风险 | 高 | 低 |
| 可解释性 | 一般 | 强 |
| 阈值可调性 | 弱 | 强 |
| 上线稳定性 | 差 | 更高 |

## 六、业务策略建议

### 6.1 前台展示

- 对普通用户只展示低侵扰式风险提示；
- 不展示过于确定的医学判断；
- High/Critical 等级需要配套免责声明与帮助入口。

### 6.2 后台审核

| 风险等级 | 建议动作 |
|----------|----------|
| None | 记录 |
| Mild | 低优先级观察 |
| Moderate | 纳入待复核列表 |
| High | 较高优先级复核 |
| Critical | 优先人工查看，可联动危机规则 |

### 6.3 告警约束

Critical 不能仅凭模型概率触发最终危机结论，应结合：

- 文本高危信号；
- 历史事件；
- 人工复核；
- 明确业务规则。

---

## 七、后续升级条件

只有在满足以下条件后，才建议重新尝试独立五级模型：

1. 获得天然五级或临床等级标签；
2. 每级样本量至少达到数百级别；
3. 引入 PHQ-9 / GAD-7 / 文本特征 / 生理特征；
4. 能明确区分“症状风险等级”和“抑郁诊断标签”；
5. 完成无泄漏验证。

---

## 八、结论

v1.22 对五级风险的正确策略不是继续训练失败的多分类模型，而是将五级风险降级为“**由二分类概率稳定映射而来的业务风险等级**”。

这是对 v1.21 真实结论的直接响应，也是当前数据条件下最稳妥、最可解释的方案。

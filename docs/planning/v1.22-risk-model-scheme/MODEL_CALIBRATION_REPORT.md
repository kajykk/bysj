## 一、校准目标

v1.22 的校准目标是将二分类模型输出的 `Depression Probability` 转换为稳定、可解释、可监控的风险分数和五级风险等级。

由于 v1.21 测试集仅 150 样本，不适合训练复杂校准器，因此 v1.22 采用规则化分段校准方案。

## 三、v1.22 风险分数映射

### 3.1 映射函数

```python
def probability_to_risk_score(p: float) -> float:
    if p < 0.05:
        return p * 200
    if p < 0.15:
        return 10 + (p - 0.05) * 100
    if p < 0.35:
        return 20 + (p - 0.15) * 100
    if p < 0.65:
        return 40 + (p - 0.35) * 83
    if p < 0.85:
        return 65 + (p - 0.65) * 100
    return min(100, 85 + (p - 0.85) * 100)
```

### 3.2 映射原则

| 原则 | 说明 |
|------|------|
| 低概率区间细分 | 便于区分正常与轻度关注 |
| 中间概率压缩 | 避免不稳定区间过度解释 |
| 高概率保守展示 | 减少概率高估带来的恐慌 |
| 输出限制 0-100 | 便于前后端统一展示 |

## 四、风险等级映射

| Risk Score | Risk Level | 业务含义 |
|------------|------------|----------|
| 0-19 | None | 无明显风险 |
| 20-39 | Mild | 轻度关注 |
| 40-59 | Moderate | 中度关注 |
| 60-79 | High | 高风险，建议复核 |
| 80-100 | Critical | 极高风险，建议优先复核 |

## 五、置信度定义

v1.22 建议同时输出 `confidence`：

```python
confidence = max(p, 1 - p)
```

解释方式：

| Confidence | 说明 |
|------------|------|
| < 0.60 | 不确定，建议谨慎解释 |
| 0.60-0.80 | 中等置信 |
| > 0.80 | 高置信，但仍非诊断 |

注意：confidence 代表模型分类确信度，不代表医学结论可靠性。

## 六、校准策略选择

| 校准方法 | v1.22 是否采用 | 原因 |
|----------|----------------|------|
| Platt Scaling | 暂不采用 | 样本过少，易过拟合 |
| Isotonic Regression | 暂不采用 | 需要更多校准集 |
| Temperature Scaling | 暂不采用 | 更适合深度模型 logits |
| 手工分段映射 | 采用 | 可解释、可控、易回滚 |
| 线上动态阈值 | 暂不采用 | 需先积累线上基线 |

---

## 七、监控与再校准触发条件

满足以下任一条件时，应重新校准阈值：

1. 线上 High/Critical 占比连续 7 天超过基线红线；
2. 人工复核中高风险误报率持续过高；
3. 新增真实标签样本超过 1,000 条；
4. 引入 PHQ-9 / GAD-7 等新标签；
5. 模型版本切换或输入特征分布明显漂移。

---

## 八、结论

v1.22 不建议使用复杂统计校准器，而应采用规则化概率分段映射。该方案适合当前小样本真实数据阶段，具有以下优点：

- 透明；
- 易解释；
- 易回滚；
- 能支持五级风险展示；
- 不会把未充分校准的概率误解释为临床严重程度。

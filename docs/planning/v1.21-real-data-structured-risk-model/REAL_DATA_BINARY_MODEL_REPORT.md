## 四、模型对比

| Metric | LogisticRegression | RandomForest | Winner |
|--------|-------------------|-------------|--------|
| Accuracy | 0.8133 | 0.9133 | RF |
| F1 | 0.6000 | 0.6829 | RF |

| Recall | 0.9130 | 0.6087 | LR |
| ROC-AUC | 0.9401 | 0.8956 | LR |
| AUPRC | 0.7914 | 0.7623 | LR |

**分析**:
- LR: 高召回 (91.3%) 但低精度 (44.7%) — 宁可误报也不漏报
- RF: 高精度 (77.8%) 但低召回 (60.9%) — 漏掉了 9/23 的抑郁患者
- **在心理健康场景中，LR 的安全属性更优**（漏报 2 人 vs 9 人）

## 五、校准样本测试

| 样本 | Expected | LR Pred | LR Proba | RF Pred | RF Proba |
|------|----------|---------|----------|---------|----------|
| 无抑郁+无焦虑+低压力 | No Depression | 0 ✅ | 0.021 | 0 ✅ | 0.058 |
| 无抑郁+有焦虑+中等压力 | No Depression | 1 ❌ | 0.651 | 0 ✅ | 0.332 |
| 抑郁+焦虑+高压力 | Depression | 1 ✅ | 1.000 | 1 ✅ | 0.800 |
| 极低-完全健康 | No Depression | 0 ✅ | 0.015 | 0 ✅ | 0.022 |

- LR 对"有焦虑但无抑郁"的样本给了 0.651 概率 → 过度敏感
- RF 在极端情况判断更保守
- 两者都能正确识别高风险和完全健康样本

## 六、vs v1.20 合成模型对比

| Metric | v1.20 Synthetic LR | v1.21 Real LR | v1.21 Real RF |
|--------|-------------------|--------------|--------------|
| Accuracy | 0.9813 | 0.8133 | 0.9133 |
| F1 | 0.9875 | 0.6000 | 0.6829 |
| Precision | ~0.98 | 0.4468 | 0.7778 |
| Recall | ~0.99 | 0.9130 | 0.6087 |
| ROC-AUC | 0.9991 | 0.9401 | 0.8956 |

### 关键结论

v1.20 的合成模型实际学到的是 heuristic 规则（训练标签本身就是 heuristic 生成的），因此指标接近完美（本质上在做自回归）。

v1.21 的真实数据模型面对的是真实世界的 Depression 标签，指标显著降低。**这说明两件事**:
1. Heuristic 规则 ≠ 真实抑郁诊断（合成模型的高指标是虚假的）
2. 用 14 个问卷特征预测临床抑郁本身就是困难的（F1=0.60-0.68 在真实场景中属于可接受但不优秀）

### 判定: **Conditional Go** ✅

- 不替换 v1.20 默认模型
- 将 v1.21 Real LR 作为实验 artifact 保留
- 线上仍使用 v1.20 模型或 fallback
- 通过 `STRUCTURED_MODEL_VERSION=v1.21_real_binary_lr` 可选择性启用

## 八、改进方向

1. **扩大数据量**: 1,000 样本偏少，特别是正样本仅 194 例
2. **特征工程**: 考虑使用 `student_depression_dataset.csv` 的 27,901 样本（需解决特征映射问题）
3. **数据增强**: 在有更多真实数据之前，可以探索合成数据+真实数据的混合训练
4. **标签质量**: Depression Yes/No 可能不如 PHQ-9 分数精细
5. **五级建模**: 如果多级标签分布更好，多分类模型可能比二分类表现更优

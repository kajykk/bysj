## 1. 测试策略

### 1.1 测试分层

| 层级 | 范围 | 工具 | 目标覆盖率 | 执行环境 |
|---|---|---|---|---|
| 单元测试 | 函数/方法/组件 | pytest / Vitest | 80% | CI / 本地 |
| 集成测试 | API/模块交互 | pytest + TestClient | 80% | CI / 本地 |
| E2E 测试 | 用户流程 | Playwright | 核心流程覆盖 | CI |

### 1.2 测试优先级

| 优先级 | 测试类型 | 说明 |
|---|---|---|
| P0 | 后端单元测试 | 核心功能必须有单元测试 |
| P0 | 后端集成测试 | API 必须有集成测试 |
| P1 | 前端单元测试 | 核心组件必须有单元测试 |
| P1 | E2E 测试 | 核心用户流程必须有 E2E 测试 |

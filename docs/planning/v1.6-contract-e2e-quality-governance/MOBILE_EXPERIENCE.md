# 移动端基础体验检查清单

## 1. 响应式布局

### 断点定义
| 断点 | 宽度 | 设备类型 |
|------|------|----------|
| xs | < 576px | 手机 |
| sm | >= 576px | 大手机 |
| md | >= 768px | 平板 |
| lg | >= 992px | 桌面 |
| xl | >= 1200px | 大屏桌面 |
| xxl | >= 1400px | 超宽屏 |

### 布局规则
- **xs**: 单列布局，全宽组件
- **sm**: 2列布局
- **md**: 3列布局
- **lg**: 4列布局
- **xl**: 5列布局

## 3. 小屏可用性

### 触摸目标
- 最小触摸区域: 44x44px
- 按钮高度: >= 48px
- 输入框高度: >= 44px
- 列表项高度: >= 56px

### 字体大小
- 标题: >= 18px
- 正文: >= 14px
- 辅助文字: >= 12px
- 最小可读: 11px

### 间距
- 页面边距: >= 16px
- 组件间距: >= 12px
- 列表项间距: >= 8px

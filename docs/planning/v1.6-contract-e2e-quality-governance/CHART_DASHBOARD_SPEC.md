# 图表与 Dashboard 展示规范

## 1. 风险趋势图 (Risk Trend Chart)

### 数据格式
```typescript
interface RiskTrendData {
  date: string        // YYYY-MM-DD
  value: number       // 0.0 - 1.0
  upperBound: number  // 预警上限
  lowerBound: number  // 安全下限
}
```

### 展示规范
- **X轴**: 日期，格式 MM/DD
- **Y轴**: 风险值，0-100% 或 0.0-1.0
- **警戒线**: 上限红色虚线，下限绿色虚线
- **数据点**: 圆形标记，hover 显示详细值
- **颜色映射**:
  - 正常: #52c41a (绿色)
  - 警告: #faad14 (橙色)
  - 危险: #f5222d (红色)

### 交互规范
- 支持缩放 (1d, 7d, 30d, 90d)
- 支持悬停查看详情
- 支持导出为图片

## 2. 模型性能面板 (Model Performance Panel)

### 指标展示
| 指标 | 格式 | 阈值 |
|------|------|------|
| Accuracy | 百分比 (xx.x%) | >= 85% 绿色，< 85% 红色 |
| Precision | 百分比 (xx.x%) | >= 80% 绿色，< 80% 红色 |
| Recall | 百分比 (xx.x%) | >= 80% 绿色，< 80% 红色 |
| F1-Score | 百分比 (xx.x%) | >= 80% 绿色，< 80% 红色 |
| ROC-AUC | 百分比 (xx.x%) | >= 0.85 绿色，< 0.85 红色 |
| AUPRC | 百分比 (xx.x%) | >= 0.70 绿色，< 0.70 红色 |

### 展示规范
- 使用卡片式布局
- 每个指标一个卡片
- 包含当前值和趋势箭头
- 趋势计算: 与上一周期对比

## 3. 系统健康监控 (System Health Monitor)

### 指标展示
| 指标 | 格式 | 阈值 |
|------|------|------|
| API 成功率 | 百分比 | >= 99% 绿色，< 95% 红色 |
| 平均延迟 | 毫秒 (xx ms) | <= 200ms 绿色，> 500ms 红色 |
| 回退率 | 百分比 | <= 2% 绿色，> 5% 红色 |
| 错误率 | 百分比 | <= 0.1% 绿色，> 1% 红色 |
| 活跃用户数 | 整数 | 无阈值 |
| 内存使用 | 百分比 | <= 80% 绿色，> 90% 红色 |

### 展示规范
- 使用仪表盘 (Gauge) 展示百分比指标
- 使用时间序列图展示趋势
- 使用状态灯展示健康状态

## 4. 数字格式统一

### 百分比
- 格式: `xx.x%`
- 示例: `85.5%`, `100.0%`
- 小数位: 1位

### 整数
- 格式: `1,234,567`
- 千分位分隔符
- 大数字: `1.2w` (万), `1.2m` (百万)

### 小数
- 格式: `0.123`
- 小数位: 3位 (科学计算)
- 小数位: 2位 (业务指标)

### 时间
- 日期: `YYYY-MM-DD`
- 时间: `HH:MM:SS`
- 相对时间: `x 分钟前`, `x 小时前`, `x 天前`

### 文件大小
- 格式: `xx.xx KB/MB/GB`
- 自动单位转换

## 5. 颜色规范

### 状态颜色
| 状态 | 颜色 | 用途 |
|------|------|------|
| 成功/正常 | #52c41a | 正常状态、通过 |
| 警告 | #faad14 | 需要注意 |
| 危险/错误 | #f5222d | 错误、失败 |
| 信息 | #1890ff | 提示、进行中 |
| 中性 | #8c8c8c | 禁用、未知 |


## 6. 响应式规范

### 断点
| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| xs | < 576px | 单列，图表堆叠 |
| sm | >= 576px | 2列，图表并排 |
| md | >= 768px | 3列，标准布局 |
| lg | >= 992px | 4列，大屏优化 |
| xl | >= 1200px | 5列，超宽屏 |

### 图表响应式
- 宽度: 100% 容器
- 高度: 固定比例 (16:9 或 4:3)
- 最小高度: 200px
- 最大高度: 500px

## 7. 加载状态

### 骨架屏
- 使用灰色占位块
- 动画: 脉冲效果
- 时长: 最长 3 秒

### 空状态
- 图标 + 文字提示
- 提供操作按钮 (如刷新)

### 错误状态
- 错误图标 + 错误信息
- 提供重试按钮

## 8. 性能规范

### 渲染性能
- 首屏渲染: < 1s
- 图表动画: < 300ms
- 数据更新: < 500ms

### 大数据量
- 数据点限制: 最多 1000 个
- 超过限制: 数据聚合/采样
- 虚拟滚动: 列表超过 100 项
